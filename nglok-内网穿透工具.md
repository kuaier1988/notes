<p>（四）启动内网穿透的命令<br>
网站上的内网穿透命令是</p>
<blockquote>
<p>ngrok HTTP 8080</p>
</blockquote>
<p>用这个命令启，发现不行，后来在网上找了个其他的命令，命令为</p>
<blockquote>
<p>ngrok http 8080 --host-header=localhost #映射本地8080端口</p>
</blockquote>
